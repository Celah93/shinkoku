## 生命保険料控除

### 新制度（平成24年1月以降の契約）

| 支払保険料 | 控除額 |
|-----------|--------|
| 2万円以下 | 全額 |
| 2万超〜4万円 | 支払額/2 + 10,000円 |
| 4万超〜8万円 | 支払額/4 + 20,000円 |
| 8万円超 | 40,000円（上限） |

### 旧制度（平成23年12月以前の契約）

| 支払保険料 | 控除額 |
|-----------|--------|
| 2.5万円以下 | 全額 |
| 2.5万超〜5万円 | 支払額/2 + 12,500円 |
| 5万超〜10万円 | 支払額/4 + 25,000円 |
| 10万円超 | 50,000円（上限） |

- 新旧合算の1区分上限: 40,000円
- 3区分（一般/介護医療/個人年金）合計上限: 120,000円

### 23歳未満の扶養親族を有する場合の特例（令和8・9年分）

所得要件を満たす23歳未満の扶養親族がいる場合、一般生命保険料の新契約に次の表を使う。

| 年間の新生命保険料 | 控除額 |
|---:|---:|
| 30,000円以下 | 全額 |
| 30,000円超〜60,000円以下 | 支払額/2 + 15,000円 |
| 60,000円超〜120,000円以下 | 支払額/4 + 30,000円 |
| 120,000円超 | 60,000円（上限） |

- 一般生命保険料の新旧合算上限: 60,000円
- 介護医療保険料と個人年金保険料は通常の表・上限を使う
- 3区分合計上限: 120,000円（変更なし）
- 16歳未満の親族も含む。他の納税者が扶養控除を取る親族も特例判定に含む
- 配偶者と青色・白色事業専従者は対象外
