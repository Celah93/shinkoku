# 確定申告書B様式 記載欄（Form B Fields）

確定申告書B（第一表・第二表）の各欄と、対応するデータの対応表。
`calc_income_tax` / `calc_deductions` の出力値を各欄に正しくマッピングするための参照資料。

### 税額の計算

| 欄番号 | 項目名 | 計算方法 |
|--------|--------|----------|
| ㉛ | 課税される所得金額 | ⑫ − ㉚（1,000円未満切り捨て） |
| ㉜ | 上の㉛に対する税額 | 速算表を適用 |
| ㉝ | 配当控除 | 該当する場合 |
| ㉟ | 住宅借入金等特別控除 | housing_loan_credit |
| ㊱〜㊳ | 政党等寄附金等特別控除 | `public_interest_donation_credit` + `npo_donation_credit` + `political_donation_credit` |
| ㊸ | 差引所得税額 | ㉜から税額控除を差し引く（マイナスの場合は0） |
| ㊹ | 復興特別所得税額 | ㊸ × 2.1% |
| ㊺ | 所得税及び復興特別所得税の額 | ㊸ + ㊹ |
| ㊻ | 源泉徴収税額 | 給与・事業・その他所得の源泉徴収税額合計 |
| ㊼ | 申告納税額 | ㊺ − ㊻（納付の場合は100円未満切り捨て） |
| ㊽ | 予定納税額 | 第1期分 + 第2期分 |
| ㊾ | 第3期分の税額（納付） | ㊼ − ㊽がプラスの場合 |
| ㊿ | 還付される税金 | ㊼ − ㊽がマイナスの場合（絶対値） |

㊱〜㊳は一つの「政党等寄附金等特別控除」欄の欄番号表示であり、3区分の合計を記載する。区分別の内訳は、各計算明細書と第二表の「特例適用条文等」で区別する。

### 寄附金調整結果と計算明細書

申告へ使う確定値は `calc_income_tax` の出力を正とする。`calc_deductions` が返す寄附金の所得控除・税額控除は方式選択前の中間候補である。

| shinkoku の出力 | 計算明細書・申告書での意味 |
|---|---|
| `donation_adjustment.income_deduction.original_amount` | 所得控除を選んだ寄附金額の合計。第一表㉙と第二表「寄附金控除に関する事項」の元額 |
| `donation_adjustment.income_deduction.eligible_amount` | 総所得金額等40%枠を適用した所得控除対象額 |
| `donation_adjustment.income_deduction.threshold_amount` | 所得控除で先に使う2,000円の足切り額 |
| `donation_adjustment.income_deduction.final_amount` | 第一表㉙の寄附金控除額 |
| 各区分の `original_amount` | 各計算明細書①の寄附金額 |
| 各区分の `eligible_amount` | 前段区分の寄附金額を差し引いた40%枠適用後の対象額 |
| 各区分の `threshold_amount` | 前段区分で消費した分を差し引いた2,000円の残額 |
| 各区分の `formula_amount` | 控除率適用後、100円未満を切り捨てた算式額 |
| 各区分の `tax_credit_cap` | 100円未満切捨て後の所得税額25%上限。NPOは公益控除後の残枠 |
| `public_interest.final_amount` | 公益社団法人等寄附金特別控除額の計算明細書⑫ |
| `npo.final_amount` | 認定NPO法人等寄附金特別控除額の計算明細書⑬ |
| `political.final_amount` | 政党等寄附金特別控除額の計算明細書⑫ |

第二表の「特例適用条文等」には、公益は `措法41の18の3`、NPOは `措法41の18の2`、政治は `措法41の18` と記載する。

## 端数処理ルール

### 所得税

| 対象 | ルール | 法的根拠 |
|------|--------|---------|
| 各所得金額 | 円単位（端数処理なし） | — |
| 課税所得金額（㉛） | 1,000円未満切捨て | 国税通則法118条 |
| 算出税額（㉜） | 円単位（速算表の結果そのまま） | — |
| 寄附金特別控除の算式額・25%上限 | 100円未満切捨て | 租税特別措置法41条の18〜41条の18の3 |
| 復興特別所得税（㊹） | 1円未満切捨て | 復興財源確保法13条 |
| 所得税及び復興特別所得税の額（㊺） | 端数処理なし（㊸+㊹の合計） | — |
| 申告納税額（㊼）＝納付の場合 | 100円未満切捨て | 国税通則法119条 |
| 還付税額 | 1円単位（切捨てなし） | 国税通則法120条 |

### 消費税

| 対象 | ルール | 法的根拠 |
|------|--------|---------|
| 課税標準額 | 1,000円未満切捨て | 国税通則法118条 |
| 消費税額（国税） | 課税標準額 × 7.8%（標準）/ 6.24%（軽減） | 消費税法29条 |
| 差引税額 | 100円未満切捨て | 国税通則法119条 |
| 地方消費税 | 差引税額 × 22/78、100円未満切捨て | 地方税法72条の89 |
