# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/).

## [0.9.1] - 2026-07-18

### Fixed
- 仕訳更新で指定した年度が保存されず、年度またぎの訂正が黙って無視される問題を修正

### Changed
- 勤労学生控除のdocstringを年分別定数への参照に改め、改正のたびに金額記載が古くならない形へ整理

## [0.9.0] - 2026-07-18

### Added
- 寄附金控除の40%枠・2,000円足切り・所得税額25%枠を法定順序で調整する計算層を追加
- 公益社団法人等、認定NPO法人等、政党等の3区分について、所得控除・税額控除の最大8候補を比較する仕組みを追加
- 区分別の選択方式、計算明細書用の中間値、最終税額控除額を所得税計算結果へ追加

### Changed
- 公益社団法人等寄附金特別控除を認定NPO法人等から分離し、3区分を別々に計算するように変更
- 寄附金特別控除の算式額と所得税額25%上限額を100円未満切捨てに変更
- 寄附金種別をDB制約と同じ5種に限定し、不明な値を入力時に拒否するように変更
- 令和7年分の申告書欄番号と寄附金計算明細書への対応をスキル資料へ反映

## [0.8.1] - 2026-07-18

### Fixed
- 1月1日生まれの配偶者・扶養親族について、民法上の年齢到達日を12月31日として正しく判定するように修正

## [0.8.0] - 2026-07-18

### Added
- 年度ごとの課税・免税区分、消費税申告方法、簡易課税事業区分をDBへ保存し、CLIから確認・部分更新できるように対応
- 消費税計算で保存済み申告方法と事業区分を任意に照合し、照合結果を返す仕組みを追加

### Changed
- 簡易課税では事業区分1〜6の明示を必須とし、未指定時に第5種として扱う黙示補完を廃止
- 課税事業者判定と消費税申告のスキルに、年度プロファイルの確認・保存手順を追加

## [0.7.0] - 2026-07-17

### Added
- 令和8・9年分の基礎控除、給与所得控除、扶養親族・勤労学生の所得要件を年分別定数として追加

### Changed
- 所得税計算で課税年分に対応する定数を選び、未対応年分を明示的に拒否するように変更

## [0.6.12] - 2026-07-15

### Changed
- `journal-add`・`journal-batch-add`・`journal-update` で、同一内容の別取引と確認できた完全一致仕訳を `--force` 付きで登録・更新できるように変更
- 強制登録した完全一致仕訳を警告として返し、申告前の重複確認で再確認できる形に変更

## [0.6.11] - 2026-07-14

### Fixed
- CSV 取込で UTF-8（BOMあり・なし）と CP932（Windows-31J）を判別し、Excel・銀行・カード明細の拡張文字を正しく読めるように修正

## [0.6.10] - 2026-07-14

### Fixed
- CSV 金額欄を文字列全体で検証し、不正な桁区切り・二重符号・数字以外を含む値を黙って金額へ変換しないように修正
- 全角数字、通貨記号、3桁区切り、末尾の「円」、括弧・三角記号による負数など、正しい金額表記の正規化に対応

## [0.6.9] - 2026-07-14

### Fixed
- 平成19年4月1日以後に取得した資産の定額法を、法定償却率（別表第八）で計算するように修正
- 普通償却費と必要経費算入額の各段階で1円未満を切り上げるように修正

## [0.6.8] - 2026-07-14

### Fixed
- 重複仕訳検索から SQLite 3.44 以降だけで使える集約関数内 `ORDER BY` を除き、古い SQLite でも動くように修正
- 勘定科目コードの並びを Python 側で正規化し、類似度判定を安定化

## [0.6.7] - 2026-07-14

### Fixed
- 貸借対照表へ開始残高を含め、期首残高がある年度の資産・負債・純資産を正しく集計するように修正

## [0.6.6] - 2026-07-13

### Fixed
- 消費税が還付になる場合、控除不足還付税額と地方消費税の還付額を合計へ正しく反映するように修正
- 地方消費税の納付額は100円未満切捨て、還付額は1円未満切捨てとして計算するように修正
- 中間納付額が本税額を上回る場合の符号付き集計値を整理

## [0.6.5] - 2026-03-20

### Added
- ふるさと納税以外の寄附金に対応し、政治活動・認定NPO法人・公益法人等への寄附を所得税計算へ反映

### Changed
- 政治活動寄附金と認定NPO法人・公益法人等への寄附について、所得控除と税額控除を区分ごとに比較し、有利な方を自動選択

### Fixed
- 政治活動寄附金の税額控除へ総所得金額等の40%上限を適用
- 寄附金控除を計算するとき、40%上限と2,000円の足切りを寄附金の合計へ一度だけ適用するように修正

## [0.6.4] - 2026-03-20

### Fixed
- 政治活動・認定NPO法人等への寄附金税額控除へ、所得税額の25%上限を適用
- ふるさと納税とその他の寄附金を合算し、寄附金控除を一つの計算経路へ統一

## [0.6.3] - 2026-03-07

### Changed
- 税法・住宅ローン・消費税仕入税額控除・電子帳簿保存の参照資料を、用途別のコンテキストスキルへ分離
- 各スキルからの参照先を整理し、単独で読み込んでも必要な資料へ到達できる構成に変更

### Removed
- 使われていなかった国税庁仕様書ダウンロード・出力比較スクリプトを削除

## [0.6.2] - 2026-03-04

### Fixed
- 中古住宅の取得と増改築で住宅ローン控除を重複適用するとき、控除額の合計へ年間上限を適用するように修正

## [0.6.1] - 2026-03-03

### Fixed
- 住宅ローン控除の必要書類から、原則として提出不要な「住民票の写し」を削除

## [0.6.0] - 2026-03-03

### Added
- 中古住宅の取得と増改築を同じ年に行った場合の住宅ローン控除の重複適用に対応
- 複数の住宅ローン控除明細を受け取り、区分ごとの控除額と合計額を計算する CLI・モデル・計算処理を追加

## [0.5.2] - 2026-03-03

### Fixed
- 住宅ローン控除の登記事項証明書について、不動産番号を申告書へ記載した場合の添付省略を案内へ反映

## [0.5.1] - 2026-03-03

### Fixed
- e-Tax の給与所得入力を年末調整済み・年末調整なしに分け、源泉徴収票の入力欄とラベル対応を修正

## [0.5.0] - 2026-02-26

### Added
- 仕訳の訂正・削除履歴を保存して検索する監査ログを追加
- 取引先、金額範囲など電子帳簿保存法を意識した仕訳検索条件を追加
- 総勘定元帳とCSV出力を追加
- 電子帳簿保存の確認手順をまとめたスキルとシステム概要を追加

### Changed
- 仕訳へ取引先を保存し、訂正・削除前後の内容を監査ログへ残すように変更

## [0.4.0] - 2026-02-24

### Added
- `/setup` に `.gitignore` の安全確認と不足設定の追記手順を追加
- README に設定ファイル・DB・出力ファイルを Git 管理から外すための案内を追加

## [0.3.0] - 2026-02-24

### Changed

## [0.2.0] - 2026-02-24

### Added
- PDF ユーティリティ CLI（`shinkoku pdf extract-text` / `shinkoku pdf to-image`）
- pdfplumber によるテキスト抽出、pypdfium2 による PNG 画像変換
- Reading スキル（5種）に PDF ファイル対応を追加（テキスト抽出 → 画像変換フォールバック）
- CI バージョンチェックジョブ（PR 時に `marketplace.json` と `pyproject.toml` のバージョン一致を検証）

### Changed
- `import_data._extract_pdf_text()` を `tools/pdf.extract_text()` に委譲しコード重複を解消
- `plugin.json` から `version` フィールドを削除（`marketplace.json` に一元化）

## [0.1.0] - 2025-02-20

### Added
- 複式簿記の帳簿管理（仕訳 CRUD・残高試算表・損益計算書・貸借対照表）
- 所得税計算（所得控除・税額控除・復興特別所得税・住宅ローン控除）
- 消費税計算（2割特例・簡易課税・本則課税）
- ふるさと納税 CRUD・控除計算・限度額推定
- データ取込（CSV・レシート・請求書・源泉徴収票・控除証明書）
- OCR 画像読取スキル（レシート・源泉徴収票・請求書・控除証明書・支払調書）
- e-Tax 電子申告スキル（Claude in Chrome による確定申告書等作成コーナー入力代行）
- Playwright フォールバック（WSL / Linux 環境向け）
- Agent Skills（SKILL.md オープン標準）による 35+ エージェント対応
- Claude Code Plugin マニフェスト
- CI パイプライン（Ruff lint + mypy + pytest + coverage）
